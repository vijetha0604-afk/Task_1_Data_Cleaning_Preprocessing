# Task 1: Data Cleaning & Preprocessing

## Objective
Clean and preprocess the Titanic dataset for Machine Learning.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Steps Performed
- Loaded the Titanic dataset.
- Explored dataset structure.
- Checked missing values.
- Filled missing values using median and mode.
- Removed the Cabin column.
- Encoded categorical features using LabelEncoder.
- Standardized numerical features.
- Visualized outliers using boxplots.
- Removed outliers using the IQR method.
- Saved the cleaned dataset.

## Files
- task1.ipynb
- Titanic-Dataset.csv
- cleaned_titanic_dataset.csv

## Outcome
Successfully cleaned and preprocessed the Titanic dataset for Machine Learning.
